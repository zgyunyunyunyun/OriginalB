# 抖音小游戏导出流程（含微信转抖音改造）

## 1. 当前项目改造结果

已完成的平台层改造要点：

- `GameManager` 新增平台运行时配置，可在 Inspector 强制切到抖音服务。
- `DouyinStorageService` 已接入 `TTSDK.TTStorage`（失败自动回退 `PlayerPrefs`）。
- `DouyinShareService` 已接入 `TTSDK.TTShare.ShareAppMessage`。
- `DouyinAuthService` 已接入 `TTSDK.TTAccount.Login`。
- `DouyinAdService` 已接入 `TTSDK.TT.CreateRewardedVideoAd` / `CreateInterstitialAd`。
- 平台服务注册逻辑支持复用：`PlatformServiceRegistrar`。

## 2. Unity 侧配置

1. 打开 `GameManager` 组件。
2. 在 `Platform Runtime` 区域确认：
   - `Apply Platform Service Override In Game Manager = true`
   - `Platform Service Mode = ForceDouyin`
3. 在 `Douyin SDK Config` 填写：
   - `Douyin Rewarded Ad Unit Id`：抖音激励视频广告位 ID
   - `Douyin Interstitial Ad Unit Id`：抖音插屏广告位 ID
   - `Douyin Force Login`：是否强制拉起宿主登录
   - `Douyin Share Default Title/ImageUrl/Query`：默认分享参数

说明：

- 你也可以把 `Platform Service Mode` 切为 `AutoByBuildSymbols`，改为由编译宏决定平台。
- 目前默认已设置为 `ForceDouyin`，便于迁移阶段快速验证。

## 3. 编译宏切换（从微信改抖音）

路径：`Edit > Project Settings > Player > Other Settings > Scripting Define Symbols`

建议保留：

- `PLATFORM_DOUYIN`

建议移除：

- `PLATFORM_WECHAT`

如果你使用 `GameManager` 强制覆盖，也可先不依赖宏切换做联调，但正式构建建议宏也切到抖音。

## 4. 抖音导出与开发者工具

1. Build Target 选择 `WebGL`。
2. 使用 TTSDK 导出工具链导出小游戏工程（目录建议如：`OutGame/OriginalB-TT`）。
3. 在抖音开发者工具中导入导出目录。
4. 在抖音开放平台配置：
   - AppID
   - 广告位（激励视频/插屏）
   - 分享能力相关配置

## 5. 真机/IDE 启动后核验

启动日志里应看到：

- `PlatformContext=DouyinPlatformContext`
- `StorageService=DouyinStorageService`
- `ShareService=DouyinShareService`
- `AuthService=DouyinAuthService`
- `AdService=DouyinAdService`

功能回归建议：

1. 存储读写：重启后积分、每日次数是否保留。
2. 分享解锁：`ShelfSpawnManager` 按钮分享流程是否成功回调。
3. 登录：是否可获得登录结果（匿名/宿主登录）。
4. 激励广告：观看完成后是否正确发放奖励。
5. 插屏广告：是否可正常触发和关闭。

## 6. 微信转抖音对照清单

- SDK：微信 SDK -> TTSDK（已安装）
- 平台服务：`WeChat*Service` -> `Douyin*Service`
- 平台上下文：`WeChatPlatformContext` -> `DouyinPlatformContext`
- 编译宏：`PLATFORM_WECHAT` -> `PLATFORM_DOUYIN`
- 导出目录：建议区分 `OriginalB-WX` 与 `OriginalB-TT`

## 7. 常见问题

1. 分享在编辑器中失败

- 编辑器不具备小游戏宿主环境，属于正常现象。

1. 广告始终不可用

- 未配置广告位 ID，或平台侧广告位未审核通过。

1. 存档不生效

- 确认日志中 `StorageService=DouyinStorageService`。
- 确认小游戏宿主环境中 TTStorage 可用。
